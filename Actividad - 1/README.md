📌 Objetivo del proyecto

El objetivo de este repositorio es almacenar y organizar una serie de ejercicios prácticos de programación.
Estos ejercicios tienen como finalidad reforzar conceptos básicos de desarrollo, lógica de programación y uso de herramientas de control de versiones como Git y GitHub.

A través de la resolución de estos ejercicios se busca:

- Practicar la lógica de programación.

- Comprender la estructura básica de Java.

- Aplicar conceptos fundamentales de Java.

- Aprender a gestionar proyectos y versiones utilizando Git y GitHub.

⚙️ Instrucciones de ejecución

Para ejecutar este proyecto en tu computador, sigue los siguientes pasos:

1️⃣ Clonar el repositorio

Abre una terminal y ejecuta el siguiente comando:

    git clone https://github.com/jcrp002/Actividad---Ejercicios-de-pr-ctica-1.git


2️⃣ Entrar en la carpeta del proyecto

    cd Actividad---Ejercicios-de-pr-ctica-1

3️⃣ Abrir el proyecto

Puedes abrir el proyecto con cualquier editor de código, por ejemplo:

- Visual Studio Code

- IntelliJ IDEA

- Eclipse

- NetBeans

4️⃣ Ejecutar

Compila el archivo correspondiente.

Ejecuta el programa desde el IDE o desde la terminal.

Ejemplo en Java:
    
    javac NombreDelArchivo.java

    java NombreDelArchivo
